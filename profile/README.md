## 了解GaoyiPlayOS
GaoyiPlayOS是一款基于Android轻度定制的**光速虚拟机ROM**。
**目前仅支持光速虚拟机的Android 12**，采用类原生风格，移除了多项光速虚拟机的臃肿组件，提升了系统运行速度，融合多种类原生组件，完美的在虚拟机上做到了Pixel体验。
拥有多种版本选择：如精简版，谷歌美化版，谷歌经典版。并且，该系统完全免费，获取方式和安装过程简单易懂，是一款面向用户的第三方虚拟机 ROM 产品

GaoyiPlayOS支持原虚拟机系统没有的莫奈取色风格，完善了 pixel 体验，支持 pixel风格和GaoyiPlayOS类原生风格切换，提供用户更多选择，新版本采用模糊纯色控制中心，配合莫奈橘色更显美观
