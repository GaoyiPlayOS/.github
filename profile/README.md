## 了解GaoyiPlayOS
GaoyiPlayOS 是一款基于 Android 轻度定制的**光速虚拟机ROM**。
**目前仅支持光速虚拟机的**，采用类原生风格，移除了多项光速虚拟机的臃肿组件，提升了系统运行速度，融合多种类原生组件，完美的在虚拟机上做到了 Pixel 体验。
拥有多种版本选择：
- 精简版
- 谷歌美化版
- 谷歌经典版

并且，该系统**完全免费**，获取方式和安装过程简单易懂，是一款面向用户的第三方虚拟机 ROM 产品。

GaoyiPlayOS 支持原虚拟机系统没有的莫奈取色 (Monet Engine)，完善了 Pixel 体验，新版本采用高斯模糊纯色控制中心，配合莫奈取色更显美观。
